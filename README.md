# Mirror & Shade Bug Fix

> **Disclaimer:** This project is 100% LLM-generated.

A BepInEx plugin that fixes the most annoying bug in **Solar Expanse** where solar mirrors and shades stop working after loading a save file.

## Installation

1. **Ensure BepInEx is installed** in your Solar Expanse game directory.
   - If not, download from: https://github.com/BepInEx/BepInEx/releases

2. **Copy the DLL** to your game's plugins folder:
   ```
   Solar Expanse/BepInEx/plugins/mirrorshade-bugfix/MirrorShadeBugFix.dll
   ```

## Compatibility

- **Game version:** Solar Expanse 0.26.4.29.11 BETA
- **BepInEx version:** 5.4.x or later
